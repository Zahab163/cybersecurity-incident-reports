## 📖 About This Repo

This repository documents my progress in cybersecurity incident analysis as part of the [Networks and Network Security](https://www.coursera.org/learn/networks-and-network-security/) course on Coursera.  

Each report is based on simulated scenarios provided in the course and demonstrates:
- Practical use of network traffic analysis tools (e.g., tcpdump)
- Identification of impacted protocols and services (DNS, ICMP, HTTPS)
- Structured incident reporting for technical and non-technical audiences
- Clear communication of findings and actionable next steps

By organizing these reports here, I’m building a portfolio that reflects both my technical skills and my ability to document and communicate cybersecurity incidents professionally.



⚠️ Disclaimer: All scenarios are fictional training exercises, not real client data.
